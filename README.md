# PROYEK WEB
## OSIS SMK BINA INFORMATIKA 2025 - 2025  

Website Osis SMK Bina Informatika ini adalah website organisasi sekolah yang dibangun untuk mempermudah pengguna (user dan admin) dalam mengakses, mencari, dan mengelola data baik sekolah maupun osis secara online. Website ini bersifat dinamis, terhubung ke database, dan terdiri dari fitur-fitur yang sesuai dengan kebutuhan.
